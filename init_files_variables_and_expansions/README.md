# Assignments

For these assignments we were tasked to create scripts using expansion commands and variable manipulation. Below you will find the links to each file with a brief explanation.

[0-alias](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/0-alias)(creates an alias)

[10-love_exponent_breath](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/10-love_exponent_breath)(display the results of a variable to the power of another variable)

[11-binary_to_decimal](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/11-binary_to_decimal)(converts a number from base 2 to base 10)

[12-combinations](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/12-combinations)(prints all possible combinations of two letters, except oo)

[13-print_float](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/13-print_float)(prints a number with two decimal places, followed by a new line)

[14-decimal_to_hexadecimal](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/14-decimal_to_hexadecimal)(converts a number from base 10 to base 16)

[15-rot13](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/15-rot13)(encodes and decodes text using the rot13 encryption)

[16-odd](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/16-odd)(prints the odd numbered line)

[17-water_and_stir](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/17-water_and_stir)(adds the two numbers stored in the environment variables WATER and STIR and prints the result.)

[1-hello_you](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/1-hello_you)(prints hello user, where user is the current Linux user)

[2-path](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/2-path)(adds a line to the PATH variable)

[3-paths](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/3-paths)(counts the number of directories in the PATH)

[4-global_variables](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/4-global_variables)(lists environment variables)

[5-local_variables](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/5-local_variables)(lists all local variables and environment variables, and functions)

[6-create_local_variable](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/6-create_local_variable)(creates a new local variable)

[7-create_global_variable](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/7-create_global_variable)(creates a new global variable)

[8-true_knowledge](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/8-true_knowledge)(prints the result of the addition of 128 with the value stored in the environment variable TRUEKNOWLEDGE, followed by a new line)

[9-divide_and_rul](https://github.com/Katrolli/holbertonschool-shell/blob/main/init_files_variables_and_expansions/9-divide_and_rule)(prints the result of POWER divided by DIVIDE, followed by a new line)
